Experiment 5 - There Once Was a Man From Verdun

Making limerick

Description

The goal of this project was to try and generate theoretically infinite limericks using a combination of a database of rhyming words and a grammar defined with tracery

Technical

The structure of a limerick is defined using a tracery grammar, based on Young Person of Smyrna by Edward Lear. To find the rhymes, a call to the DataMuse API is made, so that there are truly massive number of possible rhymes the limerick may choose from

Reflection

I am reasonably content with how this project turned out. To be honest, I was doubtful I could get it working at all, so creating a load of nonsense limericks that follow the rhyme scheme and sound good to say out loud was far better than what I had anticipated. There is something a bit funny there. I chose the title before I actually started work on the project, back when I hoped I might be able to generate limericks that make sense. Now, however, the title is even more fitting, as a punchline without the setup fits perfectly with a bunch of nonsensical limericks. A high was when I got that first set of rhymes done, because the limerick I wrote using those rhymes pretty much came to me when I first read them. That convinced me that I might actually be able to make something that worked. A low was when I realized the “machine learning association” thing the database I was accessing was using was not nearly as good as advertised. That was kind of my only hope to make the limericks make sense, so having it not work was disappointing.
